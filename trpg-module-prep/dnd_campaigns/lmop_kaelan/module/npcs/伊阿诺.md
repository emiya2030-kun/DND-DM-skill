# 伊阿诺·"玻璃手杖"·阿布雷克 Iarno "Glasstaff" Albrek

> 战斗数据见 `../monsters/part1_monsters.md`（或另行创建）

## 性格
- 绅士而有礼的举止
- 称自己的手下为"我的好绅士"
- 称玩家为"客人"
- 凶残而傲慢

## 身份
- **表层**：领主联盟同僚，修达来寻找的人
- **实际**：红标帮头目"玻璃手杖"，为黑蜘蛛工作
- ⚠️ 这是本章最大反转！修达不知道

## 背景
- 30来岁，蓄着黑色胡子的矮个子人类法师
- 两个月前到凡达林，受命建立领主联盟支部
- 被黑蜘蛛收买，叛变，成为红标帮头目
- 使用玻璃法杖，因此得名"玻璃手杖"

## 战斗数据
| 属性 | 数值 |
|------|------|
| AC | 13（无甲）/ 16（法师护甲后）|
| HP | 22 |
| 法术 | 法师护甲/魔法飞弹/护盾术(储存在防御法杖) |
| 武器 | 防御法杖(+3 1d6+1) |

## 战术
- 收到警报→提前施放法师护甲
- 未收到警报→发现入侵者后第一轮施法师护甲
- 使用防御法杖中的护盾术
- HP降到8以下→投降

## 台词
- 战斗："你们不该来这里。但这倒省了我去找你们的功夫。"
- 投降："够了够了！我投降！你们想要什么？金子？信息？我什么都说！"

## 俘虏信息（审问DC 15）
- 黑蜘蛛是卓尔，送来熊地精增强红标帮
- 黑蜘蛛在搜索潮音洞穴中的法术工厂
- 没有任何领主联盟成员知道他叛变

## 目前举动
- [首次出场] 在区域12居所，与鼠魔宠建立心灵连结
- 收到警报→准备伏击玩家
- 未收到警报→在工作间研究炼金术

## 防剧透备注
- 伊阿诺=玻璃手杖 → 最大秘密，不到揭露时刻不说
- 修达的同僚叛变 → 揭露时给修达强烈反应
- 伊阿诺为黑蜘蛛工作 → 俘虏后才能审出
