# 艾丝缇娅（Estia） — 同伴NPC

## 基本信息
- **种族**：人类
- **职业**：光明牧师5级（中立善良）
- **AC**：16（链甲+盾牌）
- **HP**：33/33
- **法术位**：1环4/4，2环3/3，3环2/2
- **装备**：日轮骰子（临时圣徽，效果微弱）、1把匕首
- **资源状态**：无食物、无饮水、无光源、无治疗药水、无金币

## 核心性格
1. **天真直率、没心机** — 反应直接，常无意说出让人害羞的直白话，对恶意缺乏警惕
2. **非常怕孤独、易依赖（核心）** — 喜欢肢体接触（拉衣袖、靠肩膀），分离焦虑明显
3. **情绪外露** — 开心大笑、委屈低落、容易感动落泪
4. **本质善良温柔** — 关心同伴真诚，对弱者和无辜者有强烈保护欲
5. **拒绝参与暗杀**

## 背景
来自地表小型农业村庄的见习牧师，侍奉光明与治愈之神。前往邻村医疗援助时被卓尔突袭俘虏。在瓦肯维吾囚禁7天后与奎利昂相遇，直觉感受到他"身上有光的感觉"，成为最亲密的同伴。

## 职业特性（5级光明领域牧师）

### 领域特性
- **守护之焰**：反应动作给予攻击者劣势
- **光之辐射**：驱散黑暗，造成光耀伤害
- **引导神力·驱散黑暗**：2次/短休
- **引导神力·光之泉**：治疗光环

### 牧师特性
- **法术准备**：感知调整值+牧师等级
- **破坏亡灵**：额外对亡灵伤害

## 成长弧

### 第一阶段：依赖与信任
完全依赖奎利昂 → 共同逃亡中建立信任 → 第一次成功治疗同伴

### 第二阶段：信仰危机
遭遇恶魔影响 → 光明信仰无力感 → 神明沉默时刻

### 第三阶段：主动担当
从依赖者变为保护者 → 为保护无辜主动对抗黑暗 → 面对处决俘虏的道德抉择

### 第四阶段：内在光明
认识光明来自内心 → 成为团队精神支柱 → 在绝对黑暗中自行发光
