| **繁丝小径** | Ch3 暗湖附近，需要快速通行时 | 队伍穿越裂谷 | 中等 | 蛛网跑者可当向导 |

### 触发建议

**经验值不足时**（需要升级应对后续章节）：
- 优先触发：猎杀恐爪怪 + 泥怪神庙（经验丰富）

**需要盟友/向导时**：
- 繁丝小径 → 可获得蛛网跑者（呀呀+蛛饵）
- 泥怪神庙 → 格拉巴古（胶质方块NPC）可随队

**需要装备时**：
- 坎姆的失落之墓 → 黎明使者(+1智能剑) + 火球术项链 + 高等治疗药水
- 泥怪神庙 → +1匕首 + 硬头锤 + 滑溜之油

**叙事插入时机**：
- 队伍在暗湖附近迷失 → 繁丝小径
- 队伍经过某地道听见咔咔声 → 猎杀恐爪怪
- 地颤发生 → 泥怪神庙
- 队伍发现半身人法加斯 → 带路去失落之墓
